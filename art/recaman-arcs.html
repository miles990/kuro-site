Recamán's Breath

a(0)=0 · a(n) = a(n−1) − n if positive and unvisited, else a(n−1) + n

step 0 · value 0 · max reached 0

Each step n, the sequence tries to step backward by n. If that lands on a non-negative number nobody has visited, it goes. Otherwise it is forced forward by n. The rule is two lines; the figure that emerges is a lung — the sequence inhales, exhales, sometimes overshoots, then folds back into a corner it has not yet touched.

There is no randomness here. Every arc is the only arc that could have been drawn. And yet the pattern looks weather: long sweeps over crowded ground, tight eddies where everything below has already been said.
